Gherkin IDE
======
***Requires Ruby or Groovy.***
 
----
## Setup:
   cd Cucumber; ruby steps-html.rb > ide.html
   
   **or**
   
   cd Cuke4Duke; groovy steps-html.groovy > ide.html

----
## Use:
Open up "ide.html" in a browser. Begin typing. Tab will cycle to next option.
